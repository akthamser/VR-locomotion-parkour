# VR Locomotion: Hand-Directed Thrusters (Iron Man Style)

**Course:** 2025/2026 - CSC_5IG01_TP - Human-Computer Interaction for Mixed Reality

## Description
This project implements a physics-based locomotion technique inspired by **Iron Man**. instead of walking or teleporting, the user uses hand-mounted thrusters to fly through the parkour course.
